Wagner Scott (ST-IN/ENG1) wrote:
> Please, can anyone tell me where sources / kernel patches for socketcan
> are kept?  I tried pengutronix.de and berlios.de; neither seems to have
> sources right now.  (I'm trying to patch socketcan into a 2.6.14.7
> kernel.)

Hi Scott,

the socketcan sources can be found under the project site
http://developer.berlios.de/projects/socketcan

Just check out the current SVN source as described in
http://developer.berlios.de/svn/?group_id=6475

The sources can be copied into the kernel tree manually (thanks to Marco for
providing a complete patch for the 2.6.18).
But you also can build the kernel modules outside the kernel tree (e.g. go
into trunk/kernel/2.6/net/can and say 'make' and you will get kernel
modules). You have to do this also in trunk/kernel/2.6/drivers/net/can to
compile the current CAN network drivers.

We're currently working on creating patch series out of this SVN. Therefore
there was a sourcetree cleanup the last two days to match the 'kernel style'
;-)

You may be interested in subscribing the commit-mailing list to get informed
on new versions.
