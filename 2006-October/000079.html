Hi Scott,

the Pengutronix branch is derived from a mid 2004 branch of a Kernel 2.4 
socketcan. The socketcan core has been improved in many cases since 
then. Especially due to many discussions on several mailing lists.

Currently we're working to bring the Pengutronix-branch up-to-date to 
the current berlios development.

BUT the CAN-netdevice-drivers do not distinguish that much. We changed 
the structure for the can_frame - this should be the only real 
difference for the CAN-netdevice-driver. So you can it use together with 
the berlios socketcan with minimum effort.

Btw. feel free to ask Robert on this topic ...

Best regards,
Oliver
