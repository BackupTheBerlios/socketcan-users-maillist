Hello Ewgenij,

please get the latest PEAK Linux driver (v6.20) from their website.

The driver includes some good documentation (PDF).

After compiling the driver with netdev-support (default on Kernels >= 2.6.25) a can0 device will appear in /proc/pcan after plugging in the adapter.

Only the bitrate setting is a bit different, as you still need the chardev-entry for that:

E.g. 

echo "i 0x4914 e" > /dev/pcanusb0

will set the adapter to 100kBit/s

The default bitrate is 500 kBit/s.
