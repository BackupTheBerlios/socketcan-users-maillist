I use the PCAN-USB devices almost everyday for testing with the pcan
linux driver 6.20. The socketcan implementation works fine.
The bitrate setting via the chardev is a bit anoying, that's why I
pass the desired bitrate as module parameter:

/etc/modprobe.d/pcan.conf:
options pcan bitrate=0x011c
