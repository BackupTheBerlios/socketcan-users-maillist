Dear SocketCAN mailing list subscribers,

i want to inform you about a new mailing list focusing the development, 
the interfaces and the configuration of CAN network devices:

https://lists.berlios.de/mailman/listinfo/socketcan-drivers

As the discussions on the socketcan-core mailing list should focus on 
the CAN networklayer (which is quite stable now) and how to write CAN 
protocols inside the Protocol Family PF_CAN, i would like to concentrate 
the driver(layer) development discussions (about CAN controllers, 
ioctls, configuration interfaces, interrupt handlings, etc.) on a 
seperate socketcan-drivers mailing list.

This should it make easier for SocketCAN members to focus their personal 
fields of interest.

See the SocketCAN mailing list overview with updated topic descriptions:
http://developer.berlios.de/mail/?group_id=6475

Best regards & keep in touch,
Oliver
