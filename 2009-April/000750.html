2.6.25 would also be enough :-)

CONFIG_CAN=m
CONFIG_CAN_RAW=m
CONFIG_CAN_BCM=m

#
# CAN Device Drivers
#
CONFIG_CAN_VCAN=m

Must be enabled.

After 'modprobe vcan' you can create virtual CAN interfaces for first tests
(see
http://svn.berlios.de/svnroot/repos/socketcan/trunk/kernel/2.6/Documentation/networking/can.txt
section 6.4)

Then try the stuff from the SVN

http://svn.berlios.de/svnroot/repos/socketcan/trunk/can-utils/

(best: check out the SVN and work in your private tree)

You can send and receive CAN Messages with the provided tools on the vcan
interfaces then to get a first impression.
