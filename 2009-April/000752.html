A Debian Standard Linux Kernel has all this enabled by default.

When you build your Kernel on your own:

make menuconfig

And enable all the above stuff in the 'Networking' section.

----

To test if your current kernel works, try:

modprobe vcan

ip link add dev vcan0 type vcan

ifconfig vcan0 up


If you can run the 'ifconfig' without problems, you are fine.
