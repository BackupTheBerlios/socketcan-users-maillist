Hi Sven,

the best (and always up-to-date) list for supported CAN hardware for 
SocketCAN can be found in the Kconfig of the CAN driver development 
directory:

http://svn.berlios.de/svnroot/repos/socketcan/trunk/kernel/2.6/drivers/net/can/Kconfig

Additional the hardware from PEAK System 
(http://www.peak-system.com/linux/index.htm) supports the netdevice 
driver model used in SocketCAN.

Btw. the Beckhoff FC5102 seems to be a dedicated CANopen PCI card that 
has a separate processor to handle CANopen directly on the card. This is 
a very specialized approach. I don't know if it's worth the effort to 
write a SocketCAN driver for this CANopen dedicated hardware ?!?

If you like to use SocketCAN and all the supported SocketCAN hardware 
for CANopen you might also take a look on www.canfestival.org ...
