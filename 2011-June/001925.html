Since I'm developer, I might assume things that you may not be familiar
with.

These are the high level steps:

* grab the ubuntu kernel source of your system
* apply the config to it
* modify the bittiming calculation setting ($ make menuconfig or whatever)
* make (takes some time)
* try with $ insmod drivers/net/can/can_dev.ko (or can-dev.ko, don't know right now).

Now, you have the fixed modules.

Since I'm not familiar with ubuntu, I don't know how to install the kernel
& modules in 'ubuntu compliant way'.
