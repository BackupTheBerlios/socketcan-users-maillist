Hi Sven,

there are so called 'error frames' that allow to filter for 'faulty
frames' as far as the CAN driver is supporting the creation of these
error frames in error conditions.

Please look into ...

trunk/kernel/2.6/include/linux/can/error.h for the description of error
frames

trunk/test/tst-err.c for a sample error printing programm.
