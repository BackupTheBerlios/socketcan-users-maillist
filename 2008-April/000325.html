Your DENX tree seems OK. To make clear what has been merged into the
DENX GIT tree "linux-2.6-denx" (see http://www.denx.de/en/Software/GIT):

Into kernel version >= 2.6.24:

 http://svn.berlios.de/svnroot/repos/socketcan/trunk/kernel/2.6/

 This is what you are using.

Into kernel version > 2.6.24:

 http://svn.berlios.de/svnroot/repos/socketcan/branches/cannetdriver26/

 The CAN core is already part of the mainstream kernel and the new CAN
 device interface is used. This allows to manage CAN controller
 settings like bitrate via sysfs file-system. It might be worth to
 switch to that version.
