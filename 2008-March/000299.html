Fabrizio Garetto wrote:
> Hello,
>    I'm trying to compile socketcan trunk with powerpc-linux-gnu-gcc
> using the command
> 
>   make CC=powerpc-linux-gnu-gcc
> KERNELDIR=/home/user/trunk/kernel/linux-2.6.22.1-lite5200B
> 
> and this is the result:
> 
>    CC [M]  /home/user/socketcan/kernel/2.6/drivers/net/can/vcan.o
>    cc1: error: unrecognized command line option "-mregparm=3"
>    cc1: error: unrecognized command line option "-march=i586"
>    cc1: error: unrecognized command line option "-maccumulate-outgoing-args"
>    make[3]: *** [/home/user/socketcan/kernel/2.6/drivers/net/can/vcan.o] Error 1
>    make[2]: *** [/home/user/socketcan/kernel/2.6/drivers/net/can] Error 2
>    make[1]: *** [_module_/home/user/socketcan/kernel/2.6] Error 2
>    make[1]: Leaving directory `/home/user/trunk/kernel/linux-2.6.22.1-lite5200B'
>    make: *** [modules] Error 2
> 
> I looked for those options inside the various Makefiles but I couldn't
> find any of them.
> Do you know where is the problem and how to fix it?

Try

  $ make ARCH=ppc KERNELDIR=<your-kernel> \
         CROSS_COMPILE=powerpc-linux-gnu- \
         CONFIG_CAN_MSCAN=m CONFIG_CAN_MPC52XX=m

Wolfgang.
