Hi all,

i added the CAN ISO-TP transport protocol for discussions and ongoing 
development to the SocketCAN SVN (rev866).

Even though this code is tested and settled for two month now it still 
comes with absolute no warranty (of course :) and has ALPHA state due to 
the possibility that we might change parts of the socket API caused by 
discussions on this list.

A brief introduction into the implementation, the tools and the things 
you need to tweak in your system to have fun with CAN ISO-TP can be 
found in trunk/README.isotp or here:

http://svn.berlios.de/svnroot/repos/socketcan/trunk/README.isotp

If you would like to have that modem-dialup-feeling back, the tool 
'isotptun' creates an IP tunnel over ISO-TP PDUs over CAN for you. It's 
quite funny to e.g. log in via ssh on CAN ...

Feedback is most welcome!

Regards,
Oliver
