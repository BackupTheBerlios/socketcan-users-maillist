Hi,

According to the MPC5200B users manual section 19.7.5 (Clock System), if 
higher frequencies are set, the internal clock source should be set to 
SYS_XTAL_IN, which is the internal clock source of 33MHz, because of 
higher accuracy than the PLL driven IPB bus clock.
On the other hand in MSCAN it is always set to the IPB clock. Is there 
some objection against using the system clock ?
