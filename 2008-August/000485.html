Thanks! It finally worked with the following configuration:

options i82527-iomem irq=4,6 base=0xD0000,0xD0100 speed=125,125
irq_mode=2,2 force_dmc=1,1 clk=8000000,8000000

(by default the clock value was set to 16MHz, and that was wrong)
