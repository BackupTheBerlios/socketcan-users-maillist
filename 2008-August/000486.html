Hi Alexey,

congrats to get it working. I was working with the driver a long time 
ago, but i remember that setting the clock to 8MHz and to use 
force_dmc=1 was also my setup. Sorry for not getting to this earlier ...

Btw. the module parameters 'irq_mode', 'force_dmc' and 'clk' are no 
parameter arrays and can only be set once for the driver.

The correct representation of your setup must be:

options i82527-iomem irq=4,6 base=0xD0000,0xD0100 speed=125,125 irq_mode=2 force_dmc=1 clk=8000000


Have fun!
