The order is defined by the file "series" but using "quilt" is not
difficult. Try to get "quilt" for your Linux distribution, unfold the
archive into the top directory of the Socket-CAN trunk and apply the
patches as shown:

$ cd <path>/trunk
$ tar xjf socketcan-sja1000dev-r572.tar.bz2
$ export QUILT_PATCHES=quilt_patches
$ quilt push -a
