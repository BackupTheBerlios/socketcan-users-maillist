Hello,

since I wanted to use BCM sockets for a couple of application we are
developing at my company, I had to find out how they actually work.

I found the German LaTeX documents inside the SVN repository and I
translated them in English. I put the files on a website and I can keep
them there forever, but I think they could also be put with their older
german brothers in the source repository.

Before the links, some notes:
- There certainly are lots of typos and errors. The translation has to
  be considered an early draft. Corrections and patches are welcome,
  especially by someone who knows the code and can check if the
  semantics are described correctly.
- I removed the Hardware chapter because it was outdated, I can re-add
  it later if there is interest. Perhaps it should be part of can.txt
  in the kernel sources ?
- I cut also the Transmission Protocols chapter, as that functionality
  is not availble in current kernels (right?). Again, can be added.
- There is a figure with some German text in it. It will have to wait
  until I have access to an X server and xfig...

And now the links:
Sources: http://www.brownhat.org/docs/socketcan/llcf_docs_sources.tar.gz
HTML: http://www.brownhat.org/docs/socketcan/llcf-api.html
PDF: http://www.brownhat.org/docs/socketcan/llcf-api.pdf

Let me know what you think, I hope this is useful to someone
else besides me...

Thanks,
Daniele
