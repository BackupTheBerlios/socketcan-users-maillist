Tried compiling on my new 64 bit machine running Ubuntu. Code needs to 
be made clean for both 32 and 64 bits.

I would suggest using int16_t, int32_t, int64_t, uint16_t, uint32_t, 
uint64_t to erase all doubt of how big your ints are.
