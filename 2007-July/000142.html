Uwe.Vogt at de.huettinger.com wrote:
>
> Hi all,
>
> I am using a PEAK PCAN-USB dongle as a netdev device under openSUSE
> 10.2 and will write some CANopen programs using the socketCAN interface.
>
> Is it possible to use the defined ioctl requests like
> SIOC[GS]CANBAUDRATE, SIOC[GS]CANMODE; etc.?
>

Hi Uwe,

currently the SocketCAN IOCTLs are not supported with the PEAK-driver
and we are discussing about the correct netdev configuration interface
in the Linux Kernel.
If you want to change the baudrates, etc. you have to follow the PEAK
documentation: Run pcan_make_devices to create also char-device inodes
and invoke something like 'echo "i 0x4914 e" > /dev/pcan32' to set the
USB-Adaptor to 100kBit/s. See the PEAK PDF documentation for details.

Have fun!
Oliver
